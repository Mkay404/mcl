# Security Policy

## Reporting Security Issues

If you discover a security vulnerability in My Campus Library, please report it to:

[spectra01s@gmail.com](mailto:spectra01s@gmail.com)

- Include a detailed description of the vulnerability
- Provide steps to reproduce the issue
- Specify the version of the application where the issue was found

## Security Updates

- We regularly release updates to address security issues
- Always use the latest version of the application
- Security patches are backported to the last stable release

## Secure Development

- All code changes undergo security review
- Dependencies are regularly updated
- We follow secure coding best practices

## Responsible Disclosure

- We will acknowledge your report within 48 hours
- You will receive updates on the progress of the fix
- Your report will be kept confidential
- Public disclosure will be coordinated with you
